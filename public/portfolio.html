Raycasting Wolfenstein 3D

I created a Wolfenstein 3D style level using C and SDL.

Wolfenstein 3D - Raycasting

I wrote a program that uses Raycasting to create a demo of a game similar to the classic Wolfenstein 3D. The aim was to learn and understand how classic games were created from scratch. I created the project for web using p5.js and for the desktop using SDL and C programming language.

    Technologies used:
  • C Programming Language for the desktop version
  • Javascript and p5.js for the web version
  • SDL for windowing
  • upng - an external library for loading textures

OpenGL for Windows

Here are some small projects I created for Windows using OpenGL for learning.

OpenGL for Windows

These programs were all created using OpenGL and Win32 SDK, using the C programming language. The aim was to study and understand OpenGL and Windows programming on the native level to ensure clarity. These programs were created using both core (programmable pipeline) and compatibility (fixed function pipelines) modes.

    Technologies used:
  • C programming language
  • Win32 SDK
  • OpenGL FFP and 4.5
I have attempted to implement all major features of OpenGL (Phong Lighting, Textures, Render to Buffer, Model Loading, etc), and have als studied the usage of all available shaders (tessellation, geometry, vertex and fragment)
